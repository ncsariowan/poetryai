About PoetryAI

PoetryAI is a website that generates poems using an AI model. PoetryAI was created for Francis Parker School's 2020 Poetry Week in San Diego.

The model uses an RNN trained on Thoreau.

The webpage is pure HTML/CSS with a Flask server backend and SQL backend running on Heroku App.
